Key Bindings

Selecting

ctrl W select a word
⇧ alt ← and ⇧ alt → select word left/right
⇧ ← and ⇧ → select text backward/forward
⇧ ↑ and ⇧ ↓ select text upward/downward
⇧ ⌘ ← and ⇧⌘ → select to the beginning of the line/to the end of the line

Moving Caret

⌘ ↑ and ⌘ ↓ go to the beginning/end of the document
⌘ ← and ⌘ → go to the beginning/end of the line
ctrl p and ctrl N go to previous/next line
alt ← and alt → one word left/right

Deleting

alt ⌫ delete one word backward
ctrl + ⇧ J delete current line

TextMate Useful Shortcuts

ctrl ⇧ W wraps up the selection in a HMTL tag, by default a <p>. The opening tag is selected so you can simply change the tag and TextMate will automatically match the closing tag. Awesome.
ctrl ⇧ L wraps up the selection as link, using the last link you have in the clipboard for the href attribute or a simple example.com link
ctrl B wraps up the selection in <strong>tag
⌘ / wraps up the current line–no matter where you are with the caret–as a comment; smart enough to match the language!
ctrl S prompt the incremental search, which is *absolutely* essential to jump from one point to another without losing focus
ctrl H prompts the contextual help with the W3C definitions of the property you're near with the caret

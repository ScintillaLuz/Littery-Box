The Beginner's Cheatsheet

Starting with web development it means that you probably suck at everything, just like I do. So, other than learn what makes web development possible–HTML & CSS basically–you also have to be proficient using a text editor, a version control system and the shell commands. And if you suck at all these things like I do (damn, why didn't I learn this in school?) you need to start from the very beginning, and you need to keep a file next to you until everything you do on an editor or on the terminal becomes so familiar you just do it automatically. I work with TextMate on MacOSX–the key bindings are shared, which makes it easier for me–and GIT.

Key Bindings

Selecting

ctrl+w select a word
⇧alt←and⇧alt→ select word left/right
⇧←and⇧→ select text backward/forward
⇧↑and⇧↓ select text upward/downward
⇧⌘←and⇧⌘→ select to the beginning of the line/to the end of the line

Moving Caret

⌘↑⌘↓ go to the beginning/end of the document
⌘←and⌘→ go to the beginning/end of the line
ctrl+pandctrl+n go to previous/next line
alt←andalt→ one word left/right

Deleting

alt⌫ delete one word backward
ctrl+⇧j delete current line
